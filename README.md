# SCT_DS_3

Customer Purchase Prediction using Decision Tree Classifier

Objective

Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data.

Dataset

Bank Marketing Dataset

Tools Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

Project Workflow

1. Loaded and explored the dataset.
2. Converted categorical variables into numerical values using Label Encoding.
3. Split the dataset into training and testing sets.
4. Trained a Decision Tree Classifier.
5. Evaluated model performance using accuracy score.
6. Visualized the decision tree.

Model Performance

- Accuracy: 76.3%

Key Learning Outcomes

- Data preprocessing and feature encoding.
- Building classification models using Decision Trees.
- Model evaluation and prediction.
- Decision tree visualization and interpretation.

Conclusion

The Decision Tree Classifier successfully predicted customer subscription behavior using demographic and behavioral features. This project enhanced my understanding of supervised machine learning and classification techniques.
